# assembly

This is a prototype REST AP based on the work by [sourcegraph](https://sourcegraph.com) on their example web application [thesrc](https://github.com/sourcegraph/thesrc).

# usage

To run the tests.

```
make test
```

To start the server application.

```
make run
```
